# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-01-17

### Added
- Core tracking library (`@statscode/core`)
  - SQLite database for local storage
  - Session and interaction tracking
  - Stats analyzer and certificate generator
- Badge system (`@statscode/badges`)
  - 12+ premium badge definitions
  - Tiered progression (Bronze to Diamond)
  - Badge checker with custom criteria
- Plugin for Claude Code (`@statscode/plugin-claude-code`)
- Plugin for Codex (`@statscode/plugin-codex`)
- Plugin for Antigravity (`@statscode/plugin-antigravity`)
- Plugin for OpenCode (`@statscode/plugin-opencode`)
- Plugin for Cursor (`@statscode/plugin-cursor`)
- Website (`packages/web`)
  - Leaderboard page
  - Profile pages
  - Badges gallery
