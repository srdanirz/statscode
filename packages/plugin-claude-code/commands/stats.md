# Stats Command

View your StatsCode statistics including hours tracked, sessions, badges, and score.

## Usage

Just type `/stats` to see your current stats summary.

## What You'll See

```
StatsCode Stats
===============
Total Hours:     127.5h
Total Sessions:  89
Score:           4.2/5.0

By Tool:
- Claude Code:   98.2h (77%)
- Cursor:        29.3h (23%)

Badges Earned: 5
- Claude Whisperer (Gold)
- Night Owl
- Precision Coder
- Speed Runner
- Early Adopter

Last Session: 2h 15m ago
```

## Tips

- Stats are saved locally in `~/.statscode/data.sqlite`
- Use `/stats:sync` to upload to the global leaderboard
- Use `/stats:badge` to get a badge for your GitHub profile
