# Export Command

Export your complete StatsCode data as a JSON file with verification hash.

## Usage

Run this command to export your stats.

## Output

Creates a `statscode-export.json` file containing:
- User ID
- Complete stats breakdown
- All earned badges
- Verification hash (SHA-256)

## Verification

The hash can be used to verify the data hasn't been tampered with.
